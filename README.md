## Setup

At terminal, Git clone your fork:

```
git clone git@github.com:yourUsername/conf.git
```

Install packages:

```
npm install
```

## Run

Start project at [http://localhost:3000](http://localhost:3000):

```
npm start
```

## Build


```
npm build
```

## Deploy

Deployment to `gh-pages` branch:

```
npm run deploy
```
